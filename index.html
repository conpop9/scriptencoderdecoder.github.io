Week 1

  • Attended Oracle Cloud Foundation Bootcamp
  • Learning about Java and primary Setup(Eclipse)
  • Implementation of code which identifies JavaScript tags in HTML documents
  • Learning about J-Soup library and using it for faster implementation of the above program

Week 2

  • Creating a Script Data Binder java function which incorporates inputs from user regarding
    the vulnerable tags to be processed.
  • Using HashMap data structure to store the configuration data
  • Using .htm tables for user friendly experience for modifying the configurations.
  • Identifying the potentially vulnerable ways in which the tags can be formatted to avoid
    detection and encoding

Week 3

  • Creating a Script Encoder and Decoder function which reads the input files
    encodes/decodes the same and writes the processed file to output folder.
  • Using Regex function and Patter Matcher in Java to identify the tag patterns.
  • The tags which needs to be encoded/decoded will be received from the Script Data
    Binder File which processes the configuration file.

Week 4

  • Incorporating two major concepts i.e., Executor Framework and
    reading in fixed memory chunks in the java program
  • Learning multi-threading, executor framework and chunking in Java.
  • Using Executor Framework and multithreading to simultaneously process many
    files making the process time efficient.
  • Instead of reading whole Java File in one go reading them in fixed size
    blocks which makes our program memory efficient and able to handle large files

Week 5

  • Learning about Logger, the default logger and the log4j logger.
  • Learning about SVN repository and using Tortoise SVN to upload our code files into Oracle
    Server.
  • Creating tag separated html files for payloads for testing,
  • Successfully testing the code for first 500 payloads
  • Incorporating logger in the program instead of System Output Functions to log and
    store the details more efficiently.

Week 6

  • Creating tag separated HTML file for all 2600 payloads
  • Successfully testing the code for complete payloads
  • Reading about and Configuring the log4J logger to log information in log files instead
    of console.
  • Creating a separate Script Encoder Decoder Main file which provides functionality
    to the user to access the program passing the input files and features
    to be incorporated
  • Creating executable JAR files file and testing the same with the all payloads
